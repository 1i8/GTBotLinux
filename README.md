# How to build
<!---

Termux<br />
1. pkg install git nano
2. git clone https://github.com/GuckTubeYT/GTBotLinux
3. cd GTBotLinux
4. bash termux.sh
5. dont forget to edit config.json, Command = nano config.json
6. ./gtbot

Linux<br />
1. sudo apt install git nano
2. git clone https://github.com/GuckTubeYT/GTBotLinux
3. cd GTBotLinux
4. bash linux.sh
5. dont forget to edit config.json, Command = nano config.json
6. ./gtbot
